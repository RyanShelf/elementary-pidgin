Elementary-pidgin

A setup for elementary pidgin themes

View the Project on GitHub pspeter3/elementary-pidgin

  • Download ZIP File
  • Download TAR Ball
  • Fork On GitHub

This is a set of pidgin themes and scripts that setup webkit message styles as well as smiley theme, buddy icon theme and a message style.

Installation

cd ~/.purple
git clone git://github.com/pspeter3/elementary-pidgin.git
sudo elementary-pidgin/webkit.sh

After that, copy the contents of elementary-pidgin out of the directory into the parent directory.
